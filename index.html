    Freeform-TECH

    Modern Information Systems are facing substantial amount of concurrency which can cause a critical contention issue for I/O system. FreeForm is focusing in a dynamically coordinated I/O architecture for addressing some of the limitations that current parallel storage architectures are facing with very large-scale systems. The fundamental idea is to coordinate I/O accesses according to the topology/profile of the infrastructure, the load metrics, and the I/O demands of each application. To achieve that we are developing Data Transfer and Management Utilities alongside with autonomous Internet of Things Devices for monitoring purposes.
